# 功能
* 倍攻倍防
  * 倍防全图生效，倍攻仅在`演习`不生效
* 舰船装填倍率：倍率越高，装填越快
* 普通移动加速，大世界移动加速
* 移除舰炮开火动画
* 移除战斗全屏CG图：见名知意
* 突破连续作战上限
* 移除困难限制：解除舰船类型限制和数值限制
* 移除低心情警告
* 解锁全皮：包括共享皮肤和宿舍皮肤
* 反和谐：通过hook内在方法来实现，涉及舰船名称、舰船皮肤、誓约(这都被和谐过了?)
* 敌方潜艇显形
* 移除觉醒动画：为了节省时间
* 大世界强制扫荡：去除`当前并未处于安全区域内`限制
