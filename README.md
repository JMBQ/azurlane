# Azur Lane MOD MENU
* Created based on [Perseus](https://github.com/Egoistically/Perseus) and [4pii4/PiePerseus](https://github.com/4pii4/PiePerseus)
* I've refactored the code logic to fix the crashing issue
# Feature
- [x] Multiple attack ratio、Multiple defence ratio
- [x] Ship reload ratio: The higher the ratio, the faster the reload(3.0.0 or later)
- [x] fastStageMovement、worldFastStageMovement
- [x] RemoveBBAnimation: Remove ship cannon firing animation
- [x] AutoRepeatMax
- [x] RemoveHardModeStatLimit：Remove restrictions on ship types and numerical values
- [x] RemoveMoraleWarning
- [x] Skins：unlock all skins, Including share skins (3.0.0 或更高)
- [x] antiHeXie：Implemented by hooking internal methods，Involving ship names, ship skins, and Propose (3.0.0 or later)
- [x] Block the upload of some cheat data to the official (3.0.0 or later)
